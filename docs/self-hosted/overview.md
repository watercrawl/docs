# WaterCrawl Self-Hosted

A self-hosted version of WaterCrawl, a powerful web crawling and data extraction platform.

## Prerequisites

- Docker Engine 24.0.0+
- Docker Compose v2.0.0+
- At least 2GB of RAM
- 10GB of free disk space
